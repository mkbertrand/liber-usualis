Help the Liber Usualis Project

The Liber Usualis Project is still nascent and in need of much work both to fill out all the text of the Hours, and to develop the many other resources and tools which it still lacks.

The best way you can help is to contribute to the Github by submitting issues whenever you notice anything missing or incorrect, and by submitting pull requests. Our code is hosted as a Git repository on Github.

Additionally, financial contributions are most welcome. I have not yet established a donation site to make the process convenient for donors. However, I do need money. For now, please reach out personally at bertrandm@duq.edu to discuss donations.
